## Purpose
A **secure, transparent lottery** on the blockchain that manages entries, conducts draws, and distributes prizes fairly.

---

## Key Features
- **Hybrid Draw Trigger:** Time-based or ticket-cap triggers  
- **Minimum Requirements:** Ensures fair draws (min participants & tickets)  
- **Ticket System:** Multiple tickets per player; 5 unique numbers (1â€“99)  
- **Randomness:** Future-block hash reduces manipulation  
- **Prize Distribution:** Host cut configurable; winners claim prizes via pull pattern  
- **Admin Controls:** Update draw interval, min/max participants, tickets, host share  
- **Event Logs:** ParticipantJoined, LotteryWinner, PrizeClaimed
