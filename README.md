## Purpose
The Lottery Smart Contract project aims to implement a **fair, transparent, and immutable lottery** that manages entries, conducts draws, and distributes prizes securely on the blockchain.

## Key Features
- Secure random winner selection  
- Entry fee handling and prize distribution  
- Admin controls for emergency pause and withdrawal auditing  
- Event logs for all major actions for easy verification
